# Design System

## 색상 시스템

### Primary Colors
- Primary: `#1976d2` (MUI Blue)
- Secondary: `#dc004e` (MUI Pink)
- Success: `#2e7d32`
- Warning: `#ed6c02`
- Error: `#d32f2f`

## 타이포그래피

### 폰트
- 기본 폰트: Roboto
- 제목 폰트: Roboto Bold
- 코드 폰트: Fira Code, monospace

### 크기
- h1: 2.125rem (34px)
- h2: 1.5rem (24px)
- h3: 1.25rem (20px)
- body1: 1rem (16px)
- body2: 0.875rem (14px)

## 간격 시스템
- 기본 단위: 8px (MUI spacing)
- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px

- Primary: 주요 액션

- elevation: 1 (기본)
